## 1、滑动窗口的最大值

描述<br>
给出一个可能包含重复的整数数组，和一个大小为 k 的滑动窗口, 从左到右在数组中滑动这个窗口，找到数组中每个窗口内的最大值。<br>
样例<br>
给出数组 [1,2,7,7,8], 滑动窗口大小为 k = 3. 返回 [7,7,8].<br>
解释：<br>
最开始，窗口的状态如下：<br>
[|1, 2 ,7| ,7 , 8], 最大值为 7;<br>
然后窗口向右移动一位：<br>
[1, |2, 7, 7|, 8], 最大值为 7;<br>
最后窗口再向右移动一位：<br>
[1, 2, |7, 7, 8|], 最大值为 8.<br>
挑战<br>
O(n)时间，O(k)的额外空间<br>
难度：超难<br>
代码：ArrWin.java
